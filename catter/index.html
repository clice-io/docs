Quick Start
🔍

Universal CDB Generation

Generate compile_commands.json from any build system — Make, Ninja, CMake, or custom scripts

🪝

Deep Process Interception

Hooks into process creation at the OS level (LD_PRELOAD on Unix, DLL injection on Windows)

📜

JavaScript Scripting

Write custom scripts with the embedded QuickJS engine to analyze, filter, or modify build commands

🌳

Build Graph Analysis

Visualize command trees and target dependencies to understand your build structure
