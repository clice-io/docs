clice

语言服务器

下一代 C++ 语言服务器，重点关注高性能和深度代码智能。

kotatsu

基础设施库

从 clice 生态抽出的工具库，包含异步运行时、反射、编解码、类型化 IPC/LSP、选项解析和通用支持。

catter

构建拦截器

拦截构建中的子进程，捕获命令、生成 CDB、分析构建过程，并支持脚本化 patch 调用。

clore

文档生成器

面向人和 agent 的现代 C++ 文档生成器，结合 LLVM 提取和 LLM 生成。

cxx-toolchains

工具链元数据

面向机器读取的 C/C++ 工具链元数据，覆盖编译器、链接器、运行时、目标、选项、ABI、sysroot 和符号。

T

更好的模板处理

使用伪实例化器处理依赖的模板名，对于复杂模板也可以有代码补全

H

头文件上下文

支持头文件在不同源文件的上下文之间进行状态切换，当然也完全支持非自包含文件

M

模块

良好的 C++20 模块支持，从代码补全到高亮到跳转全都适配

I

内存占用更低，速度更快

优秀的异步任务调度，支持编译任务取消，缓存必要的信息，避免无意义的 CPU 浪费
